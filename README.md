# default-browser

Command line tool to set the default http handler in OSX.

## Build

```sh
swift build -c release
```

## Usage

List http handlers:

```sh
> default-browser
* firefox
google_chrome
safari
```

Change default http handler

```sh
default-browser google_chrome
```
